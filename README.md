# Lumetix — Premium Lighting & Design

A premium React + Vite + React Router website for **Lumetix**, a high-end lighting design and luxury technology brand.

## Stack

- **React 18** — UI framework
- **Vite 5** — Build tooling
- **React Router v6** — Client-side routing

## Pages

| Route | Page |
|-------|------|
| `/` | Home — hero, collections, values, showroom, press, testimonials, CTA |
| `/about` | About — philosophy, craftsmanship, team, materials & sustainability |
| `/products` | Collections — Pendant, Sconces, Floor & Table, Smart Systems |
| `/contact` | Contact — enquiry form, showroom info, trade programme |

## Design System

- **Black** `#0a0a0a` — Primary background
- **Gold** `#d4af37` — Primary accent, borders, typography
- **Dark Gray** `#1a1a1a` — Card backgrounds
- **Off-white** `#f5f5f0` — Primary text
- **Silver** `#888888` — Secondary text
- **Serif** — Cormorant Garamond (headings, editorial copy)
- **Sans** — Montserrat (navigation, labels, UI)

## Getting Started

```bash

npm run dev
```
